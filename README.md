## 1️⃣ Title

### 💳 Credit Card Consumption Prediction using Machine Learning

This project focuses on predicting the **future credit card spending behavior of customers** using machine learning techniques. By analyzing historical transaction data, demographic details, and financial behavior, the model estimates how much a customer is likely to spend on their credit card in the coming months.

Financial institutions generate large volumes of customer data every day. 📊 By applying **data analytics and machine learning**, organizations can uncover patterns in customer behavior and make smarter financial decisions.

This project demonstrates how **predictive modeling** can transform raw financial data into actionable insights for businesses.

---

## 2️⃣ Objective 🎯

The primary objective of this project is to develop a **machine learning model** capable of predicting the **average credit card consumption of customers for the next three months**.

### Key Goals of the Project

✔ Analyze customer demographic and behavioral data
✔ Identify important factors influencing credit card spending
✔ Build predictive models to estimate future spending patterns
✔ Evaluate model performance using regression metrics
✔ Generate insights to support data-driven financial decisions

By achieving these goals, banks and financial institutions can **better understand customer behavior and improve financial planning strategies**.

## 4️⃣ Data Source 📂

The dataset used in this project consists of **three different datasets**, each providing important information about customers.

### 👤 1. Customer Demographics Data

This dataset includes basic information about customers such as:

* Age
* Gender
* Income level
* Account information

Demographic data helps in understanding the **background and financial profile of customers**.

---

### 💳 2. Customer Behavior Data

This dataset captures customer financial activities, including:

* Debit card transactions
* Credit card spending
* Investment information
* Loan details
* Monthly transaction patterns

These behavioral variables help the model understand **how customers manage their finances**.

---

### 📉 3. Credit Card Consumption Data

This dataset contains the **target variable**:

`cc_cons` → Average credit card consumption of customers.

The machine learning model uses all other features to **predict this value**.

---

### 🔗 Data Integration

All datasets were merged using a **unique customer identifier (Customer ID)** to create a single dataset containing both **demographic and behavioral attributes** for each customer.
