# credit-card-consumption-prediction
Machine Learning project to predict customer credit card spending using Python and ML algorithms.
# 📊 Credit Card Consumption Prediction



# 1️⃣ Title
Credit Card Consumption Prediction using Machine Learning

This project predicts customer credit card spending using machine learning algorithms based on demographic and behavioral data.

---

# 2️⃣ Objective

The objective of this project is to build a machine learning model that predicts the **future credit card consumption of customers**.

This helps financial institutions:

- Understand customer spending behavior  
- Improve credit limit decisions  
- Identify high-value customers  
- Create targeted marketing strategies

# 4️⃣ Data Source

The dataset used in this project consists of three main files.

### 1. Customer Demographics Data
Contains information such as:

- Age
- Gender
- Income
- Account information

### 2. Customer Behavior Data
Includes:

- Debit card transactions
- Credit card usage
- Investment data
- Loan details

### 3. Credit Card Consumption Data
Contains the target variable:

`cc_cons` → Credit card consumption of customers.

The datasets were merged using **customer ID** to create the final dataset.

# 6️⃣ Business Problem

Banks and financial institutions need to understand customer spending behavior to make better financial decisions.

Without predictive analysis, banks face challenges such as:

- Incorrect credit limits
- Poor customer targeting
- Financial risk
- Inefficient marketing strategies

This machine learning model helps banks **predict customer credit card spending**, enabling better decision-making and improved customer experience.
